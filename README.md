These are the configuration files I use for school and software development. This is a work in progress.

- **CLI**
    - editor: nvim
    - file manager: ranger
    - music: ncmpcpp
    - email:
    - rss: newsboat
    - todo: plaintext file
    - cal:
    - mounter: udiskie

- **GUI**
    - wm: bspwm
    - terminal: alacritty
    - launcher: rofi
    - browser: firefox
    - notif: dunst
    - pdf viewer: zathura

    - editor: vs code
